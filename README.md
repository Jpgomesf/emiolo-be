# Backend-emiolo 

## Description

This is a backend application built with NestJS and MongoDB for the Emiolo company. It provides endpoints to manage users.

## Installation

```bash
$ git clone https://github.com/Jpgomesf/backend-emiolo.git
```

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

The following API endpoints are available:
